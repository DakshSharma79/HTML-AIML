HTML Tags

Document Structure:

[Imp:All tags are under these [<>] but does not shown.So assume it accordingly.]

!DOCTYPE html: Declares the document type and version of HTML.

html: The root element of an HTML page, enclosing all other HTML elements.

head: Contains meta-information about the HTML document, such as title, stylesheets, and scripts.

title: Specifies the title of the HTML document, displayed in the browser tab or window title bar.

body: Contains the visible content of the HTML document.

Headings and Paragraphs:

<h1>; to <h6>: Define headings of different levels, with <h1> being the most important and <h6> the least.

Text Formatting:

<strong>: Indicates strong importance, typically rendered in bold.

<em>: Indicates emphasis, typically rendered in italics.

<b>: Renders text in bold, without implying strong importance.

<i>: Renders text in italics, without implying emphasis.

<mark>: Highlights text.

<small>: Renders text in a smaller font size.

<del>: Indicates deleted text, typically rendered with a strikethrough.

<ins>: Indicates inserted text, typically rendered with an underline.

<sub>: Renders text as subscript.

<sup>: Renders text as superscript.

<br>: Inserts a single line break

.

<hr>: Creates a horizontal rule (line).

Links and Images:

<a>: Defines a hyperlink and <img>: Embeds an image.

Tables:

<table>: Defines a table.

<thead>: Groups the header content in a table.

<tbody>: Groups the body content in a table.

<tfoot>: Groups the footer content in a table.

<tr>: Defines a row in a table.

<th>: Defines a header cell in a table.

<td>: Defines a data cell in a table.
